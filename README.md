# CCBHelper
CCB月考助手

## 下载方式
在Release中下载对应版本即可。

⚠️ 第一次双击运行解压后的 Mac 软件时，大概率还会遇到 macOS 的安全弹窗：
“打不开 xxx，因为无法验证开发者。”

这是因为我没有花钱购买苹果的开发者证书对这个二进制文件进行签名（个人自制工具通常不值得花这个钱）。

解决办法：
1. 打开 Mac 的“系统设置” -> “隐私与安全性”。
2. 向下滚动，会看到一行提示：“已阻止使用 xxx，因为来自身份不明的开发者”。
3. 点击旁边的 **“仍要打开”** 按钮，输入电脑密码授权即可。(或者更快的快捷操作：按住键盘的 Control 键不放，然后鼠标点击该程序，选择“打开”，在弹出的警告框中再次点击“打开”。)

## 用法
1. 准备好题库（excel文件），确定题干、选项、正确答案分别在哪些列。
2. 打开程序
3. 输入题干、选项、正确答案分别在哪些列
4. 在弹窗中选取excel文件
5. 在弹窗中选取网页html文件（进入考试界面后，右键“另存为”，即可保存html格式网页文件）
